# Proyecto de Predicción de Precios BMW

## Objetivo general
Evaluar el rendimiento de modelos predictivos aplicados a datos de vehículos BMW para comprender los factores que influyen en su precio.

## Objetivos específicos
1. Describir las características del conjunto de datos mediante un análisis exploratorio que permita la comprensión  el comportamiento de las variables.
2. Entrenar varios modelos de regresión utilizando las variables disponibles para evaluar su capacidad predictiva.
3. Concluir cuáles características tienen mayor influencia en el precio y qué modelo resulta más adecuado para la predicción.
